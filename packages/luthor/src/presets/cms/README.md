# CMS preset

## Purpose
Structured content with stronger validation and schema rules.

## Features
- Block types mapped to content models
- Field validation and required blocks
- Media library integration hooks
- Controlled formatting and limited styles
- Deterministic output for storage

## Customizability
- Define block schema and field rules
- Restrict formatting per block type
- Custom media pickers and upload flows
- Serialization to JSON or HTML
