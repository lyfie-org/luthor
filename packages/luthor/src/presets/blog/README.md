# Blog preset

## Purpose
Great for long-form writing and publishing workflows.

## Features
- Headings, block quotes, and pull quotes
- Hero image and captions
- Embeds for video and social
- Readable typography and spacing defaults
- Draft and preview friendly structure

## Customizability
- Enable or disable embeds and media types
- Control image sizes and caption styles
- Configure article metadata fields
- Theme for blog typography and layout
